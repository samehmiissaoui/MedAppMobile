## Tech Stack

- **Framework**: React Native with Expo SDK 54
- **Language**: JavaScript
- **Navigation**: Expo Router
- **Linting**: ESLint (strict rules) + Prettier
- **Pre-commit hooks**: Husky + lint-staged
- **CI/CD**: GitHub Actions
- **Code Quality**: SonarCloud
